# 社区行为准则

## 我们的承诺

我们承诺维护一个开放、友好、安全的社区环境，为所有参与者提供平等的参与机会，不分年龄、体型、残疾情况、民族认同、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性认同和性取向。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注社区整体利益
- 对其他社区成员表现出同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像
- 人身攻击、歧视性评论或煽动性言论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中被认为不适当的行为

## 执法责任

社区管理员有责任澄清和执行本行为准则；对于不可接受的行为，将采取适当公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共场所代表社区时。

## 执行

如有滥用、骚扰或其他不可接受的行为，可联系社区管理团队。所有投诉都将得到及时、公平地审查和调查。
